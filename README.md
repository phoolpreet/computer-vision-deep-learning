## Computer Vision with Deep Learning


### Classification Models


| Model | Year | Main Idea |  Top-1 Accuracy       |
| ------| ---- | --------- | --------------------- |
| ResNet | 2015 | Residual connections  | ResNet18: ~69.8% ImageNet <br> ResNet50: ~76.0% ImageNet <br /> ResNet152: ~78.3% ImageNet |
| MobileNetV1 | 2017 | Depthwise separable convolution | ~70.6% ImageNet |
| MobileNetV2 | 2018 | Inverted residuals, linear bottlenecks | ~71.8% ImageNet |
| MobileNetV3 | 2019 |  NAS, SE blocks, <br> hard-swish activation | Small: ~67.4% ImageNet <br> Large: ~75.2% ImageNet |
| MobileNetV4 | 2024 | Universal mobile architecture, <br> hybrid CNN-attention, Mobile MQA | Small: ~74.1% ImageNet <br> Medium: ~78.3% ImageNet <br> Large: ~80.2% ImageNet |
| EfficientNet | 2019 | Compound scaling of depth, width, and resolution using MBConv + SE blocks | B0: ~77.1% ImageNet <br> B1: ~79.1% ImageNet <br> B4: ~82.9% ImageNet <br> B7: ~84.3% ImageNet |
| EfficientNetV2 | 2021 | Faster training with Fused-MBConv and progressive learning | S: ~83.9% ImageNet <br> M: ~85.1% ImageNet <br> L: ~85.7% ImageNet |
